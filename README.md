# 📚 My Library

## ✨ Overview
My Library is a reading tracker and digital bookshelf concept designed as a foundation for a future mobile-first application.

The project currently focuses on a simple web prototype that organises and displays a personal book collection. The long-term vision is to evolve this into a mobile app where users can scan physical books and automatically build a personalised reading library.

Books are visually differentiated based on reading status:
- 📘 Read books appear in full colour  
- ⚫ Unread books appear in greyscale  

This project explores early-stage UI design and product thinking for a scalable reading ecosystem inspired by platforms such as StoryGraph and Fable.

---

## 💡 Features (Current Prototype)
- Digital bookshelf layout for displaying books  
- Structured UI for organising reading data  
- Basic front-end implementation using HTML/CSS/JS  
- Foundation for a scalable reading tracker system  

---

## 📱 Vision (Future Development)
This project is intended to evolve into a mobile-first reading application with features such as:

### 📷 Book Scanning
- Scan ISBN/barcodes to automatically add books
- Retrieve book metadata (title, author, cover, etc.)

### 📚 Smart Bookshelf
- Visual bookshelf interface
- Colour-coded reading status:
  - Full colour → read books  
  - Greyscale → unread books  

### 📊 Reading Tracker
- Track reading progress
- Maintain reading history
- Monitor reading habits over time

### 🌐 Future Social Features
- Reading comparisons with friends
- Recommendations based on reading history
- Community-driven suggestions
